# Personal Portfolio (HTML/CSS/JS)

A modern, responsive personal portfolio template built with plain HTML, CSS, and JavaScript.

Features:
- Responsive layout (mobile-first)
- Sticky header with mobile navigation
- Smooth scrolling for anchor links
- Dark / Light theme toggle with persistence
- Accessible semantic markup
- Contact form with client-side validation (no backend)

How to use:
1. Download the files (index.html, styles.css, script.js).
2. Open index.html in your browser to preview locally.
3. Replace placeholder text, links, and images with your own content.
4. To publish, host on GitHub Pages, Netlify, Vercel, or any static hosting provider.

To connect the contact form:
- For quick demos use a service like Formspree, Netlify Forms, or a small serverless function.
- Replace the form submit handler in script.js with a fetch() to your API endpoint.

Customization tips:
- Update colors in :root variables in styles.css.
- Add your projects by duplicating .project-card elements in the projects section.
- Replace the hero SVG with your illustration or image.
